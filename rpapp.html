Educational Application

This is a simple static website made with HTML & CSS to showcase a potential UI design for a pizza store. Users can have an easy time seeing all the menu items displayed on screen and make their order during the store's opening times.

Features

  • Displays availability of items on menu
  • Checks opening and closing times of the store and allows you to order only when the store is open.
  • Simple and Interactive UI layout
